## Buld and Run

`make run-x-y`

where x is the day and y is the part

## Build

`make src/x-y`

where x is the day and y is the part

## Build all

`make`

## Run With custom input 

`make run-stdin-x-y < input_file`

where x is the day and y is the part

## Run Executable Directly

`src/x-y < input_file`

this does not automatically rebuild when the code changes
